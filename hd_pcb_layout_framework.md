- 高速数字PCB Layout 工程师知识框架

    - 1.1 数字电路基础（时序/抖动/同步异步）
    - 1.2 常见高速接口基础（DDR/PCIe/USB/HDMI/MIPI/Ethernet）
    - 1.3 器件与封装基础（BGA/QFN/连接器/寄生）
  - 2. 传输线与信号完整性（SI）
    - 2.1 传输线模型（阻抗/反射/延迟/损耗）
    - 2.2 串扰（NEXT/FEXT/耦合路径）
    - 2.3 差分信号（差模/共模/模式转换/skew）
    - 2.4 端接与拓扑（点对点/多点/端接策略）
    - 2.5 过孔与转接（via stub/背钻/回流过孔/参考切换）
    - 2.6 眼图/裕量/抖动（layout对margin的影响）
  - 3. PCB 叠层与阻抗控制（Stackup）
    - 3.1 叠层规划（层功能分配/参考面连续）
    - 3.2 阻抗计算（单端/差分/线宽线距/介质）
    - 3.3 材料与工艺（FR-4/低损耗/DK-DF/板厂能力）
    - 3.4 回流路径（跨分割/缝隙/回流连续性）
  - 4. 电源完整性（PI/PDN）
    - 4.1 PDN 基础（目标阻抗/频域/谐振）
    - 4.2 去耦策略（选型/摆放/连接电感最小）
    - 4.3 电源平面与电流回路（铜皮/过孔阵列/压降）
    - 4.4 SSN/地弹/噪声耦合对SI影响
  - 5. EMC/EMI/ESD
    - 5.1 机理（环路面积/共模/缝隙天线）
    - 5.2 接口防护与滤波（ESD/共模电感/π滤波）
    - 5.3 分区与隔离（数模/电源区/敏感区）
    - 5.4 屏蔽与接地（地围栏/过孔栅栏/参考处理）
  - 6. 布局（Placement）方法论
    - 6.1 功能分区与优先级
    - 6.2 关键链路布局（CPU↔DDR/SerDes↔连接器/时钟）
    - 6.3 BGA 扇出与通道规划（逃逸/via类型）
    - 6.4 可制造/可返修（间距/方向/可达性）
  - 7. 走线（Routing）规则体系
    - 7.1 规则制定（class/约束/长度/偏斜/间距/阻抗）
    - 7.2 DDR 专项（分组/等长/拓扑/参考切换）
    - 7.3 SerDes/差分对专项（过孔/参考面/损耗）
    - 7.4 时钟专项（隔离/回流/噪声敏感）
    - 7.5 换层与回流（地过孔伴随/回流连续）
  - 8. DFM/DFT/可靠性
    - 8.1 工艺约束（线距孔径/阻焊/钢网/拼板）
    - 8.2 可测性（测试点/ICT-FCT/调试便利）
    - 8.3 可靠性（爬电/热/机械应力/连接器）
  - 9. 仿真/验证与协作流程
    - 9.1 约束管理与评审（review checklist/签核）
    - 9.2 SI/PI 工具认知（会看结果/会闭环）
    - 9.3 bring-up 调试配合（示波器/探头/定位）

  - 10. EDA工具与工程化能力
    - 10.1 EDA熟练度（Allegro/Altium：约束/规则）
    - 10.2 约束/规则工程化（复用/模板/脚本）
    - 10.3 文档与交付（可追溯/变更记录）
    - 10.4 自动化（批量DRC/报表/检查脚本）
  - 11. 热设计与热可靠性
    - 11.1 热源识别与功耗/温升概念
    - 11.2 散热路径（铜皮/热过孔/内层铺铜/散热器-导热垫）
    - 11.3 热与SI/PI/可靠性耦合（温升对损耗/阻抗/器件降额）

  - 12. 封装与建库（Library）
    - 12.1 库体系（Symbol/Footprint/3D/属性）
